## 1. Product Summary
The **Phonics Flashcards App** is an offline iOS learning tool designed for parents to teach phonics interactively. It digitizes ~116 phonics titles across ~9 groups, offering half of the content for free and half behind a **one-time $9.99 unlock** using **StoreKit**.

The app does **not** require:
- Backend  
- User accounts  
- Cloud sync  
- Authentication  

All logic, content, and progress tracking happens **locally on the device**.
